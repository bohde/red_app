Running a Report

  • Select or upload your chosen set of matrices.
  • Choose the product and design choices for the report.
  • Select the functions you want included in the report.
  • You will be shown a fever chart, with a summary of the functions you chose.
  • Follow the link to download the text report.

Upload

  • Things you will need:
    • Function Component Matrix
    • Component Failure Prime Matrix
    • At least one of the following:
      • Component Failure Matrix
      • Function Failure Matrix
  • All matrices must be Excel files.
  • Upload matrices will remain on the server for an hour.
